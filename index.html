GTFS format

agency.txt One or more transit agencies that provide the data in this feed DONE
stops.txt Individual locations where vehicles pick up or drop off passengers DONE
routes.txt Transit routes DONE
trips.txt Trips for each route DONE
stop_times.txt Times that a vehicle arrives at and departs from individual stops for each trip
calendar.txt Dates for service IDs using a weekly schedule. DONE/MANUAL
calendar_dates.txt Exceptions for the service IDs defined in the calendar.txt file MANUAL
shapes.txt lines on a map DONE

https://developers.google.com/transit/gtfs/reference
